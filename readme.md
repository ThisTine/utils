## Thistine's utils

`thistine.utils` is a an API for various utilities that I use in my projects.

You can read the documentation here [https://utils.thistine.com/api/swagger/index.html](https://utils.thistine.com/api/swagger/index.html).

## API List
### QRCode API
 Api to create QRCode using URL.

## Development

- Install packages: `go mod tidy`
- Run dependency injection: `wire .`
- Run the server in development mode `air`
